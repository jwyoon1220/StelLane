Setup & Run

StelLane 시작하기

1

Java Development Kit 21 설치

StelLane은 최신 언어 스펙과 최적화를 위해 JDK 21을 필요로 합니다. 설치되어 있지 않다면 Eclipse Temurin JDK 21을 다운로드 및 설치해 주세요.

2

VLC 미디어 플레이어 런타임 준비

게임의 상징인 뮤직비디오 백그라운드 영상 재생을 위해 로컬에 VLC가 설치되어 있거나, 프로젝트 루트 경로 밑 vlc/ 디렉토리에 VLC 런타임 파일이 들어가 있어야 합니다.

3

리포지토리 클론 및 커맨드 실행

오른쪽 터미널 탭에서 상황에 맞는 명령어를 복사하여 프로젝트 루트에서 실행하세요. 게임 실행 및 패키징 빌드를 빠르게 시도할 수 있습니다.

STEL-TERMINAL
./gradlew :app:runGame
